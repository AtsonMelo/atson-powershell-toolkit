# Escopo Inicial — Atson PowerShell Toolkit

## Objetivo

Documentar automações PowerShell criadas para produtividade técnica, organização profissional e integração com serviços Microsoft.

## Módulos planejados

### 1. Microsoft Graph

Automação controlada para:

- autenticação;
- envio de e-mails revisados;
- criação de eventos no calendário;
- validação de itens enviados;
- rastreabilidade de ações.

### 2. Notas em Markdown

Scripts para:

- registrar notas rápidas;
- organizar logs;
- documentar evidências;
- manter histórico de evolução técnica.

### 3. Sincronização local

Fluxos com SMB + Robocopy para:

- sincronização manual entre PC e notebook;
- geração de logs;
- preview antes da cópia;
- prevenção de apagamento acidental.

### 4. Segurança operacional

Documentação de boas práticas para:

- evitar exposição de senhas;
- não publicar tokens;
- proteger arquivos locais;
- usar cofre criptografado quando necessário.

### 5. Transcrição de áudio

Fluxo planejado para transcrição local de áudios usando ferramentas de IA, sem expor conteúdo sensível.

## Regras do projeto

Este repositório não deve conter:

- credenciais;
- tokens;
- senhas;
- logs pessoais;
- dados sensíveis;
- caminhos privados desnecessários.

## Status

Projeto em fase inicial de organização.
