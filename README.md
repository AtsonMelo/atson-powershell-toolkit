# Atson PowerShell Toolkit

Conjunto de scripts PowerShell para automação pessoal/profissional, organização de estudos, produtividade técnica e integração com serviços Microsoft.

## Objetivo

Este projeto reúne automações desenvolvidas durante a organização do meu fluxo de trabalho técnico, com foco em:

- produtividade com PowerShell;
- organização de arquivos e notas;
- integração com Microsoft Graph;
- envio de e-mails controlados;
- criação de eventos no calendário;
- sincronização manual por rede local;
- documentação de processos;
- apoio ao plano de evolução profissional em tecnologia.

## Áreas do projeto

### Microsoft Graph

Scripts para autenticação, envio de e-mail e criação de eventos no calendário Microsoft.

### Sincronização local

Scripts baseados em Robocopy para sincronização manual entre máquinas na mesma rede local.

### Notas em Markdown

Scripts para registrar notas rápidas, logs e evidências técnicas.

### Segurança operacional

Fluxos documentados para uso seguro de cofre local, como VeraCrypt, sem expor senhas, tokens ou dados sensíveis.

## O que este projeto não contém

Este repositório não deve conter:

- senhas;
- tokens;
- arquivos pessoais sensíveis;
- logs com dados privados;
- IDs internos desnecessários;
- prints com informações confidenciais.
